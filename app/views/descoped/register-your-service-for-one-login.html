Register your service GOV.UK One Login

Use this service to submit an integration request to DWP Access Manager production environment.

This will enable you to connect up to Welcome to GOV.UK One Login.

It takes around 5 minutes to register.

The Identity and Trust Service Management Team will handle your request and update you on your service status.

{{ govukButton({ text: "Start now", href: "submit-a-registration-request", isStartButton: true }) }}

Before you start

It is essential that you have all the information you need to hand before you register a request.

You will need the following:
  • Your email
  • Shared or team email address
  • Line manager email address
  • Approver's email address
  • Your DWP service name
  • Your Teams Slack channel
  • Public key
  • Public key expiry date
  • Pass redirect URL
  • Failure redirect URL
  • Logout URL
  • Scopes
  • Claims

Important information for teams

To use this service there are some pre-requisites for teams they should be aware of:

Basic understanding of:

  • OpenID Connect standards
  • Kong API Gateway & OIDC Plugin configuration process

Access to Kong Enterprise Edition, through either:

  • Integration Exchange Platfom (a.k.a distributed kong)
  • Standalone Kong

Connectivity to DTH OL OIDC Endpoints (via HCS Transit Gateway) from client service environment(s)

  • See External Identity One Login Integration Traffic Flow Diagram

Access to test environment Orchestration UI from local machines (for testing)

  • E.g. see example here
